Readme Projeto: MONITORAMENTO MENSAL DO CONSUMO DE ÁGUA DA UFSC 2013 - 2024

Projeto organizado em 3 pastas e 3 arquivos de código em *ipybn

Pastas:
- Auxiliar: contém arquivos de auxílio na organização e apresentação do projetp
- Dados: contém duas subpastas, Origem e Produtos. A pasta Produtos contém as saídas do programa em tabelas.
- Figuras: A pasta Figuras contém as saídas do programa em figuras.


Códigos *ipybn

Projeto_dados_agua_01_introducao - apresenta a estruturação dos dados;
Projeto_dados_agua_02_gerar_pd_unico - gera um único df a partir de dados de água de planilha excel;
Projeto_dados_agua_03_graficos_e_analises - gera gráficos de colunas, linhas e boxplot dos dados de consumo e realiza análises estatísticas de tendencia e sazonalidade.
